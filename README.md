Flashriot
=========

Simple script to test multiple flash files using Flashbang.

Usage: ./flashriot.sh \<flash-files-dir\> \<flashbang-dir\>

Why?
----

Flashbang is really useful, but not so feasible if we want to test a batch of files

Requirements
------------

+ PhantomJs >= 2.0.0
+ Python2
+ Bash :P
